# Ejemplo Definición Accion Elemento
1 ngClass

permite establecer una o más clases a un elemento HTML. Una alternativa más útil es definir un objeto literal cuyas claves son nombres de clases y sus valores un valor boolean que indica si se debe activar o no la clase al elemento HTML

PCA-Angular

2 ngStyle

Esta directiva nos permite añadir propiedades de CSS dinámicamente. Es decir, un margin, padding, color, etc.

Ejemplo [ngStyle] PCA

3 ngSwitch

Para hacer que NgSwitch funcione, declare la directiva estructural NgSwitch en el elemento HTML principal. Dentro del contenedor NgSwitch, defina la declaración * ngSwitchCase , la *ngSwitchCasedeclaración se refiere a una de las posibilidades del árbol de múltiples posibilidades.
